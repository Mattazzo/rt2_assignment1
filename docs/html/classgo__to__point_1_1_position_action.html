Function to reach a goal position.

Parameters
selfis the object pointer
goalis the goal position to reach

Method of the action server, if goal is canceled, robot is stopped. Otherwise, if goal is set, it works as a finite state machine with 4 state: 0 - Rotate robot to be headed for the goal 1 - Go straight ahead 2 - Robot is in the goal point, rotate to fix the heading 3 - Goal reached, stop the robot
